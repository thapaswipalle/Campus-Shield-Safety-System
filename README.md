# CAMPUS-SHIELD
**Real-Time Women Safety & Harassment Reporting System**

> **Confidentiality Notice:** Source code and operational datasets are restricted due to a published patent application with the Indian Patent Office (**App No: 202541054534**), data privacy regulations, and compliance standards. This repository contains the system architecture and technical summary.

---

## Overview
Campus-Shield is a web and location-based real-time safety system engineered to strengthen campus security and emergency response workflows. Developed by a 3-member engineering team under faculty leadership, the platform combines geospatial tracking, low-latency emergency signaling, and incident analytics.

## Key Technical Contributions
* **Geospatial & SOS Architecture:** Engineered location-based signaling mechanisms for instant emergency alerts and spatial tracking during safety incidents.
* **Incident Analytics:** Aggregated and structured incident logs to identify high-risk zones, aiding security personnel in spatial risk assessment and patrol optimization.
* **Privacy-Preserving Data Pipeline:** Implemented data sanitization routines to anonymize user reporting logs while retaining data integrity for analytical auditing.
